Цель проекта - используя знания, полученные в ходе обучения в Яндекс.Практикум по темам "Собирание и парсинг данных в интернете" провести анализ содержимого любимой игры и построить графики и рейтинги кораблей.
Проект дополнен для последней версии игры.
Часть первая посвящена получению информации из файлов игры, ее обработке, созданию и сохранению таблиц.
Часть вторая посвящена расчету нескольких аггрегатных показателей и рисованию диаграмм.
Инструменты: pandas, beatifulsoup, python, matplotlib, seaborn.
